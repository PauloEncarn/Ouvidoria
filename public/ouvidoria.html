Informações para Hospedagem

Para hospedar este sistema em sua rede interna:

  1. Salve este código em um arquivo HTML (ex: ouvidoria.html)
  2. Instale o Node.js em seu servidor
  3. Instale as dependências: npm install
  4. Execute o servidor: node server.js
  5. Acesse via: http://seu-ip:3000
  6. Para usar a rede interna, configure no DNS interno da sua rede

Backup de Dados

Os dados são armazenados no banco SQLite (ouvidoria.db).
